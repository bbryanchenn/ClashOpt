## What ClashOpt Does

- Compares Blue vs Red team compositions.
- Evaluates win conditions and synergy.
- Estimates counter risk.
- Supports workflows for local CLI use, Discord commands, and a web UI.

## Recommended Starting Path

1. Read **Getting Started**.
2. Configure your `.env` for Discord and/or web backend.
3. Run one interface first (Discord bot or web app).
4. Validate outputs with sample comps.
